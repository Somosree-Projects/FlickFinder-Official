# FlickFinder
Movie Website

## Description
FlickFinder is a web application that allows users to browse, search, and discover movies. 

## Features
- Movie browsing and search
- User authentication (login/signup)
- Movie categories
- Movie details and information
- Responsive design

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript

## Project Structure

├── css/

│   └── movies.css
├── js/

## Setup Instructions
1. Clone the repository
2. Configure your database settings in config.php
3. Import the database schema
4. Run on your local server
